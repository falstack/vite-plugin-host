# Vite Plugin Host

--------

> A vite plugin to allow you to use the custom domain for development


### Install
```sh
$ yarn add vite-plugin-host
```

### Usage
```javascript
import { vitePluginHost } from 'vite-plugin-host'


module.exports = {
  plugins: [
    // others
    vitePluginHost()
  ]
}
```
